# 贪吃蛇 Snake

使用 Python 标准库 `tkinter` 实现的经典贪吃蛇小游戏。

## 运行方式

```bash
python main.py
```

无需安装任何第三方依赖。

## 操作说明

| 按键 | 功能 |
|------|------|
| `↑ ↓ ← →` | 控制蛇的移动方向 |
| `空格` | 暂停 / 继续游戏 |
| `R` | 重新开始 |

## 游戏规则

1. 按任意方向键开始游戏。
2. 控制蛇吃掉红色食物，每吃一个得分 +10，蛇身变长。
3. 撞到墙壁或自己的身体，游戏结束。
4. 随着得分增加，蛇的移动速度会逐渐加快。
